## What is tsim?

tsim is a GPU-accelerated quantum circuit simulator that supports fast sampling from Clifford+T circuits with Pauli noise. It uses stabilizer rank decomposition together with ZX-calculus simplification rules to efficiently simulate hundreds of qubits, as long as the circuit doesn't have too many non-Clifford instructions.

tsim feels just like [Stim](https://github.com/quantumlib/Stim), but supports non-Clifford gates including `T`, `T_DAG`, `R_X`, `R_Y`, `R_Z`, and `U3`.

## Quick Example

```python
import tsim

c = tsim.Circuit("""
    RX 0
    R 1
    T 0
    PAULI_CHANNEL_1(0.1, 0.1, 0.2) 0 1
    H 0
    CNOT 0 1
    DEPOLARIZE2(0.01) 0 1
    M 0 1
    DETECTOR rec[-1] rec[-2]
""")

detector_sampler = c.compile_detector_sampler()
samples = detector_sampler.sample(shots=100)
```
