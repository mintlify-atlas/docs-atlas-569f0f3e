## Overview

The `Circuit` class is a thin wrapper around `stim.Circuit` that extends support for non-Clifford gates like T gates and arbitrary rotations. It provides a familiar interface for building and manipulating quantum circuits.

## Constructor

### `__init__`

```python
def __init__(self, stim_program_text: str = "") -> None
```

Initialize a circuit from STIM program text.

<ParamField path="stim_program_text" type="str" default="">
  STIM program text to parse. If empty, creates an empty circuit. Supports shorthand syntax for non-Clifford gates.
</ParamField>

**Example:**

```python
from tsim import Circuit

# Create a simple circuit
circuit = Circuit('''
    H 0
    T 0
    CNOT 0 1
    M 0 1
''')

# Create an empty circuit
empty_circuit = Circuit()
```

## Building Circuits

### `append`

```python
def append(
    self,
    name: str,
    targets: Union[int, stim.GateTarget, stim.PauliString, Iterable[...]] = (),
    arg: Union[float, Iterable[float], None] = None,
    *,
    tag: str = ""
) -> None
```

Append an operation to the circuit.

<ParamField path="name" type="str" required>
  The gate name (e.g., "H", "CNOT", "T", "R_Z"). Can also be a `stim.CircuitInstruction` or `stim.CircuitRepeatBlock`.
</ParamField>

<ParamField path="targets" type="Union[int, Iterable]" default="()">
  The qubit(s) operated on by the gate. Can be a single int or an iterable of targets.
</ParamField>

<ParamField path="arg" type="Union[float, Iterable[float], None]" default="None">
  Gate parameters such as error probabilities or rotation angles.
</ParamField>

<ParamField path="tag" type="str" default="">
  A customizable string attached to the instruction.
</ParamField>

**Examples:**

<CodeGroup>

```python Basic Gates
circuit = Circuit()
circuit.append("H", 0)
circuit.append("CNOT", [0, 1])
circuit.append("M", [0, 1])
```

```python Non-Clifford Gates
circuit = Circuit()
# T gate (shorthand)
circuit.append("T", 0)
circuit.append("T_DAG", 1)

# Rotation gates
circuit.append("R_Z", 0, arg=0.25)  # R_Z(π/4)
circuit.append("R_X", 1, arg=0.5)   # R_X(π/2)

# U3 gate
circuit.append("U3", 0, arg=(0.3, 0.24, 0.49))
```

```python Noise
circuit = Circuit()
circuit.append("H", 0)
circuit.append("DEPOLARIZE1", 0, arg=0.01)
circuit.append("M", 0)
```

</CodeGroup>

## Compilation

### `compile_sampler`

```python
def compile_sampler(self, *, seed: int | None = None) -> CompiledMeasurementSampler
```

Compile the circuit into a measurement sampler.

<ParamField path="seed" type="int | None" default="None">
  Random seed for the sampler. If None, a random seed will be generated.
</ParamField>

**Returns:** A `CompiledMeasurementSampler` that can sample measurement outcomes.

**Example:**

```python
circuit = Circuit('''
    H 0
    CNOT 0 1
    M 0 1
''')

sampler = circuit.compile_sampler(seed=42)
samples = sampler.sample(shots=1000)
print(samples.shape)  # (1000, 2)
```

### `compile_detector_sampler`

```python
def compile_detector_sampler(self, *, seed: int | None = None) -> CompiledDetectorSampler
```

Compile the circuit into a detector sampler.

<ParamField path="seed" type="int | None" default="None">
  Random seed for the sampler. If None, a random seed will be generated.
</ParamField>

**Returns:** A `CompiledDetectorSampler` that can sample detectors and observables.

**Example:**

```python
circuit = Circuit('''
    R 0 1
    H 0
    CNOT 0 1
    M 0 1
    DETECTOR rec[-1]
    OBSERVABLE_INCLUDE(0) rec[-2]
''')

sampler = circuit.compile_detector_sampler(seed=42)
detectors, observables = sampler.sample(
    shots=1000,
    separate_observables=True
)
```

### `detector_error_model`

```python
def detector_error_model(
    self,
    *,
    decompose_errors: bool = False,
    flatten_loops: bool = False,
    allow_gauge_detectors: bool = False,
    approximate_disjoint_errors: bool = False,
    ignore_decomposition_failures: bool = False,
    block_decomposition_from_introducing_remnant_edges: bool = False
) -> stim.DetectorErrorModel
```

Return a detector error model describing the error processes in the circuit.

<ParamField path="decompose_errors" type="bool" default="False">
  When True, attempts to decompose composite errors into simpler errors. Not supported by tsim.
</ParamField>

<ParamField path="flatten_loops" type="bool" default="False">
  When True, the output will not contain any repeat blocks.
</ParamField>

<ParamField path="allow_gauge_detectors" type="bool" default="False">
  When False, verifies that detectors are deterministic under noiseless execution.
</ParamField>

<ParamField path="approximate_disjoint_errors" type="bool | float" default="False">
  When True or a threshold probability, approximate disjoint error components as independent.
</ParamField>

<ParamField path="ignore_decomposition_failures" type="bool" default="False">
  When True, undecomposed errors are inserted into the output instead of aborting.
</ParamField>

<ParamField path="block_decomposition_from_introducing_remnant_edges" type="bool" default="False">
  Requires both components of a decomposition to appear elsewhere in the model.
</ParamField>

**Returns:** A `stim.DetectorErrorModel` describing error mechanisms.

**Example:**

```python
circuit = Circuit('''
    H 0
    DEPOLARIZE1(0.01) 0
    M 0
    DETECTOR rec[-1]
''')

dem = circuit.detector_error_model(allow_gauge_detectors=True)
print(dem)
```

## Operators

The `Circuit` class supports various operators for combining and manipulating circuits:

<CodeGroup>

```python Addition
# Combine circuits
c1 = Circuit("H 0")
c2 = Circuit("CNOT 0 1")
c3 = c1 + c2

# In-place addition
c1 += c2
```

```python Multiplication
# Repeat circuit
c = Circuit("H 0")
repeated = c * 3  # Creates repeat block

# In-place multiplication
c *= 3
```

```python Indexing
circuit = Circuit("H 0\nX 1\nCNOT 0 1")

# Get single instruction
instr = circuit[0]  # First instruction

# Slice circuit
sliced = circuit[1:3]  # Instructions 1-2
```

```python Equality
c1 = Circuit("H 0")
c2 = Circuit("H 0")
print(c1 == c2)  # True

# Approximate equality
c1 = Circuit("DEPOLARIZE1(0.01) 0")
c2 = Circuit("DEPOLARIZE1(0.011) 0")
print(c1.approx_equals(c2, atol=0.01))  # True
```

</CodeGroup>
