# Quick Start

This guide will walk you through creating your first quantum circuit with tsim, compiling it into a sampler, and generating measurement samples.

## Your First Circuit

Let's start with a simple example that demonstrates the key features of tsim:

```python
import tsim

# Create a circuit with Clifford and non-Clifford gates
circuit = tsim.Circuit("""
    H 0
    CNOT 0 1
    T 0
    M 0 1
""")

# Compile into a measurement sampler
sampler = circuit.compile_sampler()

# Generate 100 measurement samples
samples = sampler.sample(shots=100)

print(samples)
```

<Tip>
The circuit string syntax is identical to Stim's format, making it easy to port existing Stim circuits to tsim.
</Tip>

## Step-by-Step Example

Let's build a more complete example that showcases tsim's capabilities:

<Steps>

<Step title="Import tsim">
First, import the tsim library:

```python
import tsim
import numpy as np
```
</Step>

<Step title="Create a Circuit with Non-Clifford Gates">
Build a circuit that includes T gates and rotation gates - operations not supported by pure Clifford simulators:

```python
circuit = tsim.Circuit("""
    # Initialize qubits
    RX 0
    R 1
    
    # Apply non-Clifford T gate
    T 0
    
    # Add some noise
    PAULI_CHANNEL_1(0.1, 0.1, 0.2) 0 1
    
    # Clifford operations
    H 0
    CNOT 0 1
    DEPOLARIZE2(0.01) 0 1
    
    # Measure
    M 0 1
    DETECTOR rec[-1] rec[-2]
""")
```

<Info>
The circuit includes:
- `RX` and `R`: Reset operations
- `T`: Non-Clifford π/4 phase gate
- `PAULI_CHANNEL_1`: Pauli noise channel
- `H` and `CNOT`: Standard Clifford gates
- `DEPOLARIZE2`: Two-qubit depolarizing noise
- `DETECTOR`: Annotation for error detection
</Info>
</Step>

<Step title="Compile a Detector Sampler">
For circuits with detector annotations, compile a detector sampler:

```python
detector_sampler = circuit.compile_detector_sampler()
```

This creates an optimized sampler that can efficiently generate detector samples.
</Step>

<Step title="Generate Samples">
Generate samples from your circuit:

```python
# Generate 1000 detector samples
samples = detector_sampler.sample(shots=1000)

print(f"Sample shape: {samples.shape}")
print(f"First 10 samples:\n{samples[:10]}")
```

The result is a NumPy array where each row represents one shot and each column represents a detector outcome.
</Step>

</Steps>

## Working with Rotation Gates

tsim supports arbitrary rotation gates, allowing you to simulate a broader class of quantum circuits:

```python
circuit = tsim.Circuit("""
    # Rotation around X-axis by π/2
    R_X(0.5) 0
    
    # Rotation around Y-axis by π/4
    R_Y(0.25) 1
    
    # Rotation around Z-axis by π
    R_Z(1.0) 2
    
    # General single-qubit unitary
    U3(0.5, 0.25, 0.125) 3
    
    # Measure all qubits
    M 0 1 2 3
""")

sampler = circuit.compile_sampler()
samples = sampler.sample(shots=100)
```

<Note>
All rotation angles are specified as multiples of π. For example, `R_X(0.5)` applies a rotation by 0.5π = π/2.
</Note>

## Understanding Detector Samples

One key difference from Stim is how tsim handles detector samples:

```python
# Circuit with a deterministic X error followed by measurement
circuit = tsim.Circuit("""
    X 0
    M 0
    DETECTOR rec[-1]
""")

sampler = circuit.compile_detector_sampler()
samples = sampler.sample(shots=10)

print(samples)
# Output: All True values
```

<Warning>
**Important difference from Stim**: tsim reports actual detector samples, while Stim reports detection *flips* (XORed with a noiseless reference). The same circuit in Stim would return all `False` values.
</Warning>

## Working with Observables

You can also work with observable annotations:

```python
circuit = tsim.Circuit("""
    R 0 1 2
    X 2
    M 0 1 2
    DETECTOR rec[-2]
    DETECTOR rec[-3]
    OBSERVABLE_INCLUDE(0) rec[-1]
""")

sampler = circuit.compile_detector_sampler()

# Sample detectors and observables separately
detectors, observables = sampler.sample(
    shots=100, 
    separate_observables=True
)

print(f"Detector samples shape: {detectors.shape}")
print(f"Observable samples shape: {observables.shape}")
```

## Reproducible Results

Set a random seed for reproducible sampling:

```python
circuit = tsim.Circuit("""
    H 0
    M 0
""")

# Same seed produces same samples
sampler = circuit.compile_sampler(seed=42)
samples1 = sampler.sample(shots=100)
samples2 = sampler.sample(shots=100)  # Different from samples1

# Reset with same seed
sampler = circuit.compile_sampler(seed=42)
samples3 = sampler.sample(shots=100)  # Identical to samples1
```

## Complete Example

Here's a complete example putting it all together:

```python
import tsim
import numpy as np

# Create a circuit with various gate types and noise
circuit = tsim.Circuit("""
    # Initialize with Hadamard gates
    H 0 1 2
    
    # Create entanglement
    CNOT 0 1
    CNOT 1 2
    
    # Apply T gates (non-Clifford)
    T 0
    T_DAG 1
    
    # Add rotation gates
    R_Z(0.25) 2
    
    # Add noise
    DEPOLARIZE1(0.01) 0 1 2
    
    # More gates
    H 0
    CNOT 0 1
    
    # Measure
    M 0 1 2
    
    # Add detector annotations
    DETECTOR rec[-1] rec[-2]
    DETECTOR rec[-2] rec[-3]
    OBSERVABLE_INCLUDE(0) rec[-1]
""")

# Compile the circuit
sampler = circuit.compile_detector_sampler(seed=42)

# Generate samples
samples = sampler.sample(shots=1000, append_observables=True)

print(f"Generated {len(samples)} samples")
print(f"Sample shape: {samples.shape}")
print(f"Detector firing rate: {np.mean(samples, axis=0)}")
```
